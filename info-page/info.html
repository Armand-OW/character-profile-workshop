üî§ What is HTML?

HTML is the structure of a webpage - think of it like the bones of a skeleton. It tells the browser what goes where.

You can edit things like:

  • Headings: <h1>Welcome!</h1>
  • Text content: <p>This is a paragraph.</p>
  • Images: <img src="pic.jpg" alt="Nice photo">
  • Links: <a href="https://example.com">Visit Site</a>
Notice the opening and closing tags of some of the elements, for example: <h1>Title</h1>.
üé® What is CSS?

CSS is like clothing‚Äîit styles how your webpage looks.

Examples of things you can change:

  • Text color: color: red;
  • Background color: background-color: #f0f0f0;
  • Fonts and spacing: font-size: 20px;, margin: 10px;

You will usually find CSS inside a <style> tag or in a separate main.css file.

In the main.css file, you'll see variables like --trait-background-color and --text-color being used. You can change these values to customise the look of your character profile page!
